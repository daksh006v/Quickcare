## 🚨 Problem Statement

Traveling to a new city or country is exciting, but falling ill during a trip can be highly stressful. Travelers regularly face:
- 🔍 **Lack of trusted recommendations** for hospitals or clinics.
- 💬 **Language barriers** preventing effective communication of symptoms.
- 💸 **Hidden costs** and lack of transparency on consultation fees before visiting.
- ⏱️ **Wasting precious hours** searching for the correct specialist.
- 😰 **High stress** making medical decisions in an unfamiliar environment during an emergency.

---

## 💡 Solution

**[👉 Experience the live application here](https://quickcare-rto6.onrender.com/)**

QuickCare is a centralized medical assistance platform that:
- 🗺️ **Pinpoints verified healthcare providers** based on a traveler's real-time location.
- 💳 **Provides full transparency** regarding fees, specialties, and available languages.
- 🩺 **Guides users** through a symptom-based triage system to the most suitable specialist.
- 💊 **Extends support** to pharmacy discovery and medicine comparison to complete the care loop.

## ✨ Features

- 📍 **Location-Based Discovery**: Detects GPS location to display an interactive map of nearby hospitals and clinics.
- ✅ **Verified Providers**: All healthcare providers are vetted, with patient ratings displayed for trust.
- 🧠 **AI Triage**: Users describe symptoms in plain language, and AI recommends the right specialist.
- 🔎 **Smart Filters**: Filter by distance, cost range, language, and specialty.

---

## 🛠️ Tech Stack

- **Frontend**: React 19, Vite 8, React Router v7, Tailwind CSS v3, Lucide React
- **Backend**: Node.js, Express (Currently serving the static frontend app)
- **Deployment**: Render Web Service (Coupled Architecture)

## 🚀 Getting Started

### Prerequisites
- Node.js (v18 or above)
- npm
- Git

### Installation

```bash
# Clone the repository
git clone https://github.com/daksh006v/Quickcare.git
cd Quickcare

# Install frontend dependencies
cd frontend
npm install

# Start the development server
npm run dev
```

The application will be available at `http://localhost:5173`.
